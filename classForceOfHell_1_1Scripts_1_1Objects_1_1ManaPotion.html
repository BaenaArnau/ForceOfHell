Detailed Description

Represents a mana potion area that restores mana to a player upon entry.

The ManaPotion class inherits from Area2D and provides a bobbing visual effect. When a player enters the area, a predefined amount of mana is restored to the player, up to their maximum mana, and the potion is removed from the scene. This class is typically used as a collectible or power-up in gameplay scenarios.

override void ForceOfHell.Scripts.Objects.ManaPotion._Process ( double  delta)

Updates the object's position to create a vertical bobbing effect based on the elapsed time since the last frame.

The bobbing speed and amplitude are determined by the BobSpeed and BobAmplitude properties. This method should be called every frame to maintain smooth motion.

Parameters
deltaThe time, in seconds, that has passed since the previous frame. This value influences the calculation of the bobbing motion.

Handles the event when a body enters the associated area and restores mana to the player if applicable.

This method is typically connected to an area or trigger in the scene. If the entering node is a player, the player's mana is increased by a predefined amount, up to the maximum allowed. The area is then removed from the scene.

Parameters
bodyThe node that has entered the area. If the node is a player, its mana will be increased.
