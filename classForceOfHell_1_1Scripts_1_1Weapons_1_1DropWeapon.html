Detailed Description

Represents a weapon drop area in the game that displays weapon details and manages player interactions, including random weapon assignment and granting mana upon pickup.

When a player enters the designated information area, a popup displays the weapon's details. The class assigns a random weapon from the available set and updates the player's weapon and mana when the drop is collected. The popup is positioned above the weapon sprite and is only shown to player nodes. This class is intended to be used within the Godot engine as an Area2D node.
