Detailed Description

Represents the climbing movement state of the player, managing interactions and physics during climbing.

This state handles the player's transition into climbing, updates movement based on input, and ensures proper alignment with climbing ladders. It also manages state transitions based on climbing conditions, such as initiating climbing, jumping, or returning to idle or running states.

override void ForceOfHell.Scripts.StateMachine.MovementStateMachine.States.ClimbMovementState.UpdatePhysics ( double  delta)
virtual

Updates the player's physics state during climbing, adjusting movement and animation based on input and climbing conditions.

If climbing is not permitted, the player's velocity is reset and sliding occurs. When climbing is allowed, vertical movement is determined by input, and the appropriate climbing or idle animation is set based on movement. This method should be called once per physics frame to ensure responsive climbing behavior.

Parameters
deltaThe time elapsed since the last frame, in seconds. Used to calculate movement adjustments for smooth physics updates.
