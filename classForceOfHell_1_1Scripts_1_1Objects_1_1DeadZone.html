Detailed Description

Represents an area within the scene that eliminates player characters and certain enemies upon entry.

This class is typically used to define zones that trigger elimination behavior for players and specific enemy types when they enter. After eliminating a player, the area removes itself from the scene. It is commonly connected to area triggers in game scenes to enforce boundaries or hazards.

void ForceOfHell.Scripts.Objects.DeadZone.OnBodyEntered ( Node  body)

Handles the event when a body enters the associated area and triggers the appropriate response if the body is a player.

This method is typically connected to an area or trigger in the scene. If the entering body is a player, the player will be eliminated and the area will remove itself from the scene.

Parameters
bodyThe node that has entered the area. If this node is a player, the player will be affected accordingly.
