Detailed Description

Represents a destructible box in the game world that can take damage and drop items upon destruction.

The Box class inherits from RigidBody2D and manages its own health and destruction state. When the box's health reaches zero, it triggers an item drop based on predefined probabilities and removes itself from the scene. This class is intended for use in scenarios where interactive, destructible objects are needed, such as breakable containers or obstacles. Item drops are delegated to the first level, and the box is freed after attempting to drop an item.

void ForceOfHell.Scripts.Objects.Box.RandomizeDrop ( )
private

Randomly generates and drops an item at the current position based on predefined probabilities.

This method selects an item type to drop using random chance and delegates item creation to the first level. The method frees the current object after attempting to drop an item. If the current scene is not available, no item is dropped and the object is freed immediately.
